## 오늘의 wiki 페이지 (Dataview)

```dataview
TABLE WITHOUT ID
  file.link AS Page,
  status AS Status,
  file.folder AS Section,
  tags AS Tags
FROM "docs/wiki"
WHERE date = this.file.day
  AND file.name != "index"
  AND file.name != "log"
SORT file.folder ASC, file.name ASC
```
