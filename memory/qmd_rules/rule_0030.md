# [TGL] Rule #30

**Keywords:** 밴드폭,band width,변동성,ATR,가격 파라미터,난이도,게임 밸런스,고정값,dynamic difficulty

**Rule:** 게임 내 가격 기반 파라미터(밴드폭, 변동성 임계값, 난이도)를 설정할 때 절대값(고정 %) 금지. 반드시 ATR(Average True Range) 등 최근 실제 변동성 기반 동적 값 사용. 차트는 상대적 2D 공간 — 게임 난이도는 화면 내 상대적 움직임에 연동.

**Context:** [TGL] 발생: 밴드 최대폭을 고정 가격%(±0.3%)으로 설정 → 저변동시 모두 적중/고변동시 모두 실패 → 게임 밸런스 파괴

**Result:** 위상 케이스: 금융 게임의 가격 기반 파라미터를 절대값으로 고정하면 시장 변동성과 게임 난이도가 디커플링됨. 차트는 상대적 2D 공간이므로 파라미터도 상대값(ATR 등)이어야 함.
