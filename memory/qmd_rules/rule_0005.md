# [TCL] Rule #5

**Keywords:** 퇴근 종료 마무리 끝 세션종료 QMD qmd_sessions 세션기록

**Rule:** 세션 종료 시: qmd_sessions/YYYY-MM-DD_session{N}.md에 세션 기록을 QMD 포맷으로 작성하여 저장. wesanggoon-sessions 컬렉션에 자동 인덱싱됨.

**Context:** [TCL] 원문: 세션 종료 시 프로젝트 로컬저장소에 위상군만의 QMD파일을 만들어서 해당세션의 기록을 저장

**Result:** 위상적 변환 완료 → 규칙 활성화
