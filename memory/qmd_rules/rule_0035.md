# [TGL] Rule #35

**Keywords:** overlay pointer-events chart interactive setPointerCapture

**Rule:** (1) pointer-events:none + capture-phase 리스너로 필요한 이벤트만 선택 가로채기 (2) setPointerCapture 금지 (3) useEffect deps에 자주 변하는 값 넣지 말고 ref로 안정화 (4) 상수 속성 참조 전 존재 확인 필수

**Context:** [TGL] 발생: pointer-events:auto overlay가 lightweight-charts의 네이티브 기능(실시간 업데이트, 축 스크롤, 줌)을 모두 차단. setPointerCapture가 하위 라이브러리와 충돌. undefined 상수 참조로 NaN silent failure 발생.

**Result:** 위상 케이스: UI overlay + 인터랙티브 라이브러리 이벤트 충돌
