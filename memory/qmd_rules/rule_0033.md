# [TCL] Rule #33

**Keywords:** 빌드,구현,build,implement,코드 작성,개발,빌드군

**Rule:** /build-review 스킬로 이미 구현됨. 원본: 빌드→기획군 리뷰→종일군 테스트

**Context:** [TCL] 원문: 빌드군이 구현 시 테스트코드 포함, 기획군이 코드+테스트 실행으로 기획의도 반영 검증, 티키타카 후 종일군 테스트 요청. TPB에서 실패한 기능을 MRV에서 정립할 것.

**Result:** 위상적 변환 완료 → 규칙 활성화
