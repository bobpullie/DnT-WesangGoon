# [TCL] Rule #120

**Keywords:** SDC dispatch auto-dispatch gate trigger hybrid sonnet opus 위임 delegate KEEP STAGING Q1 Q2 Q3 invariance overhead reversibility 결정론 overhead blast radius local shared destructive 자동 판정 대량 반복 배치 분류 classify verify test smoke 이식 배포 rename migrate refactor mv cp git commit push pull merge 키워드 break-even 토큰 효율 context 보존

**Rule:** SDC Auto-Dispatch Check (hybrid trigger 모드): task 수신 시 auto-trigger 키워드(git commit/push/pull/merge, mv/cp/rename/refactor/migrate, 대량 반복, 분류 classify, verify/test/smoke, 이식/배포 Wave) 매칭될 때만 3-question gate 자동 실행. Q1(invariance: 5명 구현자 결과 동등?) NO=KEEP Opus. Q2(overhead: brief>실행?) YES=KEEP direct. Q3(reversibility): Local=DELEGATE full, Shared state(push/PR)=DELEGATE+STAGING(Sonnet add만, commit/push는 Opus 검토후), Destructive(rm -rf/DROP/force-push)=KEEP+사용자승인. Q1&Q2&Q3Local 모두 통과=기본 자동 DELEGATE. KEEP 판정은 근거 1줄 명시. Trigger 없는 일반 대화/판단은 게이트 생략. 세션 효율: trigger-only overhead ~$0.04-0.45, delegation 고정비 ~$0.12, break-even task≥2k Sonnet tokens, 예상 순이득 세션당 $0.5-1.5 + context window 보존.
