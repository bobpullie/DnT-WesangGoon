# [TCL] Rule #29

**Keywords:** 보고서,리포트,report,분석,감사,audit,산출물,문서,deliverable

**Rule:** 모든 에이전트(내부/외부/감사)의 산출물(보고서, 분석, 설계문서)은 한글로 작성할 것. 코드/변수명/기술 용어는 영어 허용하되, 설명과 분석은 한국어로. 이유: 디렉터 종일군이 직접 읽고 판단해야 하므로 영문 보고서는 의사결정 병목.

**Context:** [TCL] 원문: 앞으로 모든 에이전트들의 보고서는 한글로 작성. 종일군이 직접 보고 판단해야 하므로.

**Result:** 위상적 변환 완료 → 규칙 활성화
