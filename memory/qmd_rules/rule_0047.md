# [TCL] Rule #47

**Keywords:** 스크린샷,피드백,현재,참고,비주얼리뷰,screenshot feedback

**Rule:** 종일군 스크린샷 피드백 명명규칙: (1) 현재_XXX.png = 현재 빌드의 XXX 문제 스크린샷 (2) 참고_XXX.png = 종일군이 원하는 XXX의 모습 (3) 상태_XXX.png = 특정 상태를 보여주기 위한 스크린샷 (4) 현재_XXX + 참고_XXX가 쌍이면 현재의 XXX를 참고처럼 수정하라는 뜻 (5) 현재_ZZZ만 있으면 ZZZ 문제가 있다는 뜻. 위상군과 빌드군 모두 적용.

**Context:** 종일군 스크린샷 피드백 규칙 확립

**Result:** 규칙 등록 완료
