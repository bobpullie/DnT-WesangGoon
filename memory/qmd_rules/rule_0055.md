# [TCL] Rule #55

**Keywords:** docker compose restart env environment variable .env down up recreate

**Rule:** Docker 컨테이너 환경변수 변경 시 restart가 아닌 down+up 필요. restart는 환경변수를 갱신하지 않음. 시스템 환경변수가 .env 파일보다 우선함.

**Context:** auto-registered

**Result:** pending
