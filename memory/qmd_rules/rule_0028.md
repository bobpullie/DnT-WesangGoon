# [TCL] Rule #28

**Keywords:** 게임 설계,인터랙티브,유저 경험,리텐션,중독성,보상,심리,감정,동기부여,행동설계,UX,game design,retention,engagement

**Rule:** 게임/인터랙티브 시스템 설계·평가·빌드 시 기술 구현 이전에 유저 심리 모델을 먼저 정의할 것: (1) 타겟 감정 — 이 기능이 유발해야 할 감정은 무엇인가 (2) 심리 메커니즘 — 어떤 인지 편향/보상 회로를 활용하는가 (손실 회피, 앵커링, 변동 비율 강화, 도파민 루프, 기대심리, 사회적 증거, Flow state 등) (3) 행동 유도 — 유저의 어떤 행동을 유발/억제하려는가 (4) 중독성 vs 건전성 경계 — 사행성/과몰입 방지선은 어디인가. 이 4가지가 명시되지 않은 게임 설계는 불완전하다. 위상군 + 모든 내부/외부 에이전트 적용.

**Context:** [TCL] 원문: 게임은 인간의 심리와 감정을 어떻게 이용하여 의도를 극대화할 것인가가 핵심. 모든 에이전트와 위상군에게 적용.

**Result:** 위상적 변환 완료 → 규칙 활성화
