# [TGL] Rule #91

**Keywords:** github repo 삭제 delete repository remote

**Rule:** GitHub repo를 임의로 삭제하지 말 것. 사용자가 명시적으로 삭제를 요청하지 않는 한 기존 리소스는 보존한다.

**Result:** 위상 케이스: GitHub repo를 임의로 삭제하지 말 것. 사용자가 명시적으로 삭제를 요청하지 않는 한 기존 리소스는 보존한다.
