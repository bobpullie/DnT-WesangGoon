# [TCL] Rule #50

**Keywords:** 해상도 resolution viewport 가로 landscape 화면크기 레이아웃기준 1280 720 사이드패널

**Rule:** DnT는 1280x720 가로형(Landscape) 게임이다. 디자인 기준 1280x720, CSS viewport 640-840x360-420, 차트 DPR 네이티브. 사이드패널 210px 상시 고정. 모든 UI/레이아웃/폰트는 이 기준으로 설계. 세로형 설계 금지.

**Context:** [TCL] 원문: DnT 화면 사양 확정: 1280x720 가로형 Landscape, 사이드패널 210px 상시 고정

**Result:** 위상적 변환 완료 → 규칙 활성화
