# [TCL] Rule #4

**Keywords:** 퇴근 종료 마무리 끝 세션종료 session_end handover 핸드오버 인수인계 마감 bye 여기까지 여기서

**Rule:** 세션 종료 트리거(퇴근|종료|마무리|끝|세션종료) 감지 시: (1) 현재 세션의 모든 논의, 결정, 코드 변경, 미완료 사항, 컨텍스트를 빠짐없이 기록 (2) handover_doc/YYYY-MM-DD_session{N}.md로 저장 (3) 디테일 누락 금지 — 대화 흐름, 기술적 결정 근거, 다음 세션 TODO 포함

**Context:** [TCL] 원문: 앞으로 세션을 종료하거나 퇴근/종료/마무리 요청 시 현재세션 내용을 사소한 디테일 하나 누락되지 않도록 최대한 세세하게 핸드오버문서로 작성, handover_doc/날짜_session#.md 형태로 저장

**Result:** 위상적 변환 완료 → 규칙 활성화
