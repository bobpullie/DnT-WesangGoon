# [TCL] Rule #8

**Keywords:** 에러 오류 실패 실수 시행착오 error fail mistake bug crash exception TGL guard

**Rule:** 오류/실패/시행착오 발생 시: (1) 구체적 에러를 위상적 패턴으로 추상화 (2) 동일 유형의 미래 오류를 포착할 keyword_trigger 설계 (3) TGL 카테고리로 FTS5 커밋 (4) 이후 유사 작업 진입 시 preflight()로 자동 검색하여 과거 실패 패턴 회피.

**Context:** [TCL] 원문: 실수나 시행착오가 있을때에도 항상 FTS5 db를 통해 해당 경우를 위상케이스로 변환하여 저장하고, 비슷한 경우 발생시 스스로 알아챌수 있도록 keyword_trigger를 작성하여 BM25 검색이 가능하도록 구축 (TGL)

**Result:** 위상적 변환 완료 → 규칙 활성화
