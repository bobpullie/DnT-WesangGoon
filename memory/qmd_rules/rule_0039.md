# [TGL] Rule #39

**Keywords:** chart lightweight-charts overlay coordinate priceScale timeScale autoScale 차트 좌표 오버레이 세로축 줌

**Rule:** lightweight-charts 차트 조작 통합 가드레일:
(1) 시점 고정: timestamp(openTime) 사용, 배열 인덱스/길이 금지 (슬라이딩 윈도우 드리프트 방지)
(2) 좌표 변환: chartCoordinates.ts 유틸리티 필수 사용, 직접 timeToCoordinate/priceToCoordinate 금지
(3) 오버레이 좌표: timeScale().width() 사용, container.getBoundingClientRect() 금지
(4) 세로축 프레이밍: autoScale:true + scaleMargins 적용 → 2프레임 후 autoScale:false 전환 (2단계 필수)
(5) 줌/정렬 변경 시: autoScale이 활성화되어 세로 패닝 잠금 → 명시적으로 autoScale:false 해제 필요

**Context:** [TGL] 발생: klines sliding window(MAX_KLINES) + setData() resets logical index from 0 every update

**Result:** 위상 케이스: finite buffer + full-reset API makes absolute index collapse to relative
