# [TGL] Rule #100

**Keywords:** 관찰 시스템 자기 재귀 self-reference 루프 IGNORE_PATTERNS jsonl hook 자기 출력 monitoring

**Rule:** [TGL-M / L2] 관찰 시스템(monitoring/logging hook)이 자기 출력을 자기 입력으로 인식하면 재귀적 self-trigger 루프가 발생한다. 관찰자와 피관찰자의 분리가 깨지면 jsonl이 노이즈로 가득 차 신호 신뢰도 0이 된다.
FORBIDDEN: 관찰 hook이 자기 jsonl/log 파일 cat/read 명령, 자기 alert 출력의 echo, 또는 자기 검증 명령 자체를 분석 대상에 포함
REQUIRED: 1) IGNORE_PATTERNS에 자기 로그 경로 cat|tail|head 정규식 명시. 2) hook 검증 명령은 sys.argv 또는 환경변수 표식으로 자기 호출 식별. 3) 매번 jsonl 변경 후 자기 trigger 발동률 모니터링. 4) 관찰자/피관찰자 분리는 위상학적 필수 — 같은 채널에서 둘이 만나면 안 된다.
