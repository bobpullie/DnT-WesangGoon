# [TCL] Rule #36

**Keywords:** 조작부 입력 이벤트 마우스 터치 드래그 클릭 패닝 줌 오버레이 interaction UX control

**Rule:** 조작부(입력 UI) 기획 시 반드시 이벤트 분배 설계 포함: (1) 입력 레이어 구분 (2) 입력 수단별 동작 매핑(좌클릭/우클릭/휠/터치) (3) 경합 해결 규칙 (4) 하위 레이어 보존 목록 (5) 경계 동작 (6) 상태별 입력 변화. 기획군 하네스 §14에 상세 체크리스트 있음.

**Context:** [TCL] 원문: 앞으로 기획군이 조작부 관련 기능을 문서에 포함할 때는 꼭 이벤트 설계까지 고려해서 문서를 작성할 것

**Result:** 위상적 변환 완료 → 규칙 활성화
