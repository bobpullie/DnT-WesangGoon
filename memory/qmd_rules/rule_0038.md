# [TGL] Rule #38

**Keywords:** 빌드군 구현,파일 재작성,이전 수정,Protected Code,퇴행,롤백

**Rule:** [⚠️ 관리군 검토요청: PROTECTED_CODE.md 참조 — 현재 DnT에 이 파일이 존재하는지 확인 필요] 빌드 지시 시 docs/process/PROTECTED_CODE.md 참조를 포함. 빌드군 BUILD 프로토콜에 Protected Code 확인 단계 추가. 기획군 리뷰에서 퇴행 체크 필수.

**Context:** [TGL] 발생: 빌드군이 파일을 재작성할 때 이전 세션의 버그 수정 사항이 무의식적으로 롤백됨 (TimeHighlight 시간축 고정)

**Result:** 위상 케이스: 서브에이전트 구현 시 기존 수정 사항 비가시성 → 퇴행 발생
