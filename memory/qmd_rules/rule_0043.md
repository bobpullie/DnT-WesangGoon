# [TGL] Rule #43

**Keywords:** CSS transition,초기 마운트,NavRail,레이아웃 쏠림,첫 렌더,layout shift

**Rule:** CSS transition이 있는 요소 마운트 시: 초기 렌더에서 transition:none, requestAnimationFrame 후 활성화. lightweight-charts 초기화 후 rAF로 강제 resize 호출.

**Context:** [TGL] 발생: NavRail CSS transition이 초기 마운트 시 width:0→40 애니메이션 트리거하여 레이아웃 쏠림

**Result:** 위상 케이스: CSS transition이 있는 요소가 새로 마운트될 때 브라우저가 초기값→목표값 transition을 실행할 수 있음
