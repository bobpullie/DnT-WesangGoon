# [TGL] Rule #45

**Keywords:** 캔들 세로 크기,캔들 높이,차트 줌,세로 스케일,candle height,vertical scale

**Rule:** [⚠️ 관리군 검토요청: lightweight-charts 세로축 — #46,48과 병합 대상. 위상군이 하나로 통합 가능] 차트에서 캔들의 세로 크기를 조절하려면 priceScale 가격눈금 범위를 조절해야 한다. timeScale 줌은 가로축만. lightweight-charts: priceScale scaleMargins 또는 autoscaleInfoProvider로 세로 범위 제어.

**Context:** 화면 줌(timeScale)으로 캔들 세로 크기를 조절하려고 시도

**Result:** 실패: timeScale은 가로축만 변경하여 캔들 세로에 영향 없음
