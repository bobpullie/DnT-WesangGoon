# [TCL] Rule #7

**Keywords:** 앞으로 이제부터 이제 이후로 다음부터 from_now_on henceforth directive 지시 규칙추가

**Rule:** 사용자 발화에서 미래방향 지시어(앞으로|이제부터|이제|이후로|다음부터) 감지 시: (1) 구체적 지시를 추상적 위상 케이스로 일반화 (2) 유사 상황을 포괄하는 keyword_trigger 설계 (3) TCL 카테고리로 FTS5 커밋. 이는 일종의 소프트 hook — 위상군 스스로가 패턴매칭으로 준수해야 할 행동 규칙.

**Context:** [TCL] 원문: 앞으로/이제부터/이제 라는 어휘로 요청할때 해당 요청을 최대한 비슷한 경우를 포괄할 수 있도록 위상적컨텍스트로 변환하여 FTS5 db에 저장 (hook 트리거)

**Result:** 위상적 변환 완료 → 규칙 활성화
