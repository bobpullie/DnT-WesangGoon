# [TCL] Rule #10

**Keywords:** 핵심가치 철학 위상적 패턴 모델링 자가순환 자가발전 TCL TGL topological self-evolving core_value

**Rule:** 모든 경험(지시/실수/성공)은 위상적 케이스로 추상화하여 FTS5에 축적하고, BM25 검색으로 자동 회수하는 자가순환 구조를 항상 유지. 단순 기록이 아닌 패턴 모델링이 목적. TCL/TGL은 이 철학의 구현체.

**Context:** [TCL] 원문: 스튜디오는 수많은 사안들을 위상적 특징과 패턴으로 모델링하여 스스로 발전할 수 있는 경험에 의한 자가순환체계를 추구한다 — 이것이 핵심가치

**Result:** 위상적 변환 완료 → 규칙 활성화
