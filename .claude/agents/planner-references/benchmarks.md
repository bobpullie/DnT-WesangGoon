## 리텐션 핵심 공식

- **AARRR**: Acquisition → Activation → Retention → Revenue → Referral
- **LTV** = ARPDAU x (1 / (1 - D1_retention)) 또는 리텐션 커브 적분
- **참여도 점수** = f(세션 길이, 빈도, 깊이, 소셜 인터랙션)

## 코어 루프 vs 메타 루프

- **코어 루프**: 1세션 내 반복 (예측→결과→보상→재예측). 30초~5분 사이클
- **메타 루프**: 세션 간 진행 (시즌 랭킹→보상→다음 시즌). 일~월 사이클
- 코어 루프가 재미없으면 메타 루프가 의미 없고, 메타 루프가 없으면 D30이 붕괴

---

## FTUE 베스트 프랙티스

- 10초 이내 첫 액션 (특히 TMA 환경)
- 30초 이내 코어 루프 1회 완료
- 첫 세션에서 "아하 모먼트" 도달
- 설명 최소화, 행동으로 학습 (Show, Don't Tell)

## TMA (텔레그램 미니앱) 생태계 데이터

### TMA 유저 행동
- 세션 길이: **1-3분** (일반 모바일 대비 극단적으로 짧음)
- 일 세션 수: **5-15회** (높은 빈도, 낮은 깊이)
- 온보딩: **10초 이내** 첫 액션 도달 필수
- 진입 장벽: 설치 없음 (텔레그램 내 바로 실행)

### 성공 사례 분석

| 앱 | 핵심 메커니즘 | 바이럴 계수 |
|----|------------|:---------:|
| Notcoin | 탭 투 언 (극단적 단순) | k=2.5+ |
| Catizen | 머지 + 고양이 (감정 결합) | k=1.8 |
| Hamster Kombat | 경영 시뮬 + 데일리 | k=3.0+ |

### 실패 패턴
- 복잡한 온보딩 (30초 이상 설명)
- 모바일 게임을 그대로 포팅 (세션 길이 불일치)
- 바이럴 메커니즘 없이 오가닉만 의존
- 멀티어카운팅 방치 (20-40% 가짜 유저)

### 수익화
- **Stars 결제**: 30% 수수료 (Apple/Google과 동일)
- **ARPU**: $0.05-0.30 (일반 모바일 대비 매우 낮음)
- **바이럴 계수**: k=1.5-3.0이 일반적. 바이럴이 곧 UA 전략

### 멀티어카운팅
- TMA에서 20-40%가 멀티 계정
- 보상 기반 시스템에서 어뷰징 심각
- 대응: 행동 패턴 분석, 점진적 보상, 소셜 검증

## 예측 게임 전문 지식

### 라운드 길이 vs 스킬 시그널

| 라운드 길이 | 스킬 비중 | 특성 |
|:-----------:|:---------:|------|
| 1분 | ~0% | 순수 노이즈, 동전 던지기. 도박에 가까움 |
| 3분 | ~2-3% | 극미세 시그널, 스킬 착각 유발 위험 |
| 5분 | ~5% | 미세 모멘텀 감지 가능, 스캘핑 영역 |
| 15분 | ~15% | 단기 추세 인식 가능, 기술분석 유효 시작점 |
| 1시간 | ~30% | 뉴스/이벤트 반영, 분석 유의미 |
| 24시간 | ~50% | 펀더멘털 반영, Superforecaster 영역 |

**기획 원칙:** 라운드가 짧을수록 "재미있지만 스킬이 아닌" 것을 명확히 커뮤니케이션해야 윤리적.

### 듀얼 캐던스 패턴
- **고빈도 라운드** (5-15분): 참여 유지, 즉각 만족, 낮은 스킬 시그널
- **저빈도 라운드** (1-24시간): 분석 깊이, 높은 스킬 시그널, 진정한 예측력 측정
- 두 캐던스를 결합하면 "빠른 재미"와 "깊은 성취"를 동시 충족

### 채점 시스템
- **Brier Score**: (예측확률 - 실제결과)^2. 0이 완벽, 1이 최악. 적절한 채점 규칙(proper scoring rule)
- **밴드 기반 채점**: 가격이 예측 밴드 내에 들어오면 성공. 밴드폭 = f(ATR, 시간프레임)
- **신뢰도 가중 채점**: 유저가 신뢰도를 표시하면 가중치 적용. 캘리브레이션 훈련 유도
- **캘리브레이션** (Tetlock Superforecasting): "70% 확신" 예측이 실제 70% 적중해야 잘 캘리브레이션된 것

### 안티갬블링 설계 ("Three Removes")
도박 삼요소(대가 + 우연 + 상금) 중 최소 하나를 제거:
1. **대가 제거**: 무료 참여, 가상 화폐만 사용
2. **우연 제거**: 스킬 시그널이 유의미한 긴 라운드 + 분석 도구 제공
3. **상금 제거**: 실물 환전 불가, 게임 내 보상만
