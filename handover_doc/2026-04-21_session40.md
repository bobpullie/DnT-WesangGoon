작업 특성상 **코드 변경 없음**. wiki 신규 페이지 1건 Draft + log 업데이트 1건.

### 2. 종일군 오해 교정 (P0 — 개념 정렬)

**오해:** "Obsidian 에 글을 쓰면 에이전트가 그걸 읽어 프롬프트로 받는다."

**실제 (Karpathy 원문):** 대화 → LLM 편집 → Obsidian 뷰어 — **단방향**. Obsidian 은 입력 창이 아니라 출력 뷰어 + 인간 큐레이션 창.

**양방향 변형 설명:**
- 위상군은 이미 **부분 양방향** 운영 중:
  - SessionStart hook + `handover_doc/CURRENT_STATE.md` 자동 주입
  - UserPromptSubmit hook + TEMS preflight (BM25/dense 매칭)
  - 명시적 `Read` tool 요청
- 완전 양방향(Obsidian → 자동 프롬프트 주입) 은 `inbox.md` 감시 hook 또는 Copilot/Smart Composer 플러그인 필요 — **현재 미구현**
