  - (a) Templater의 user script 기능 — JS로 외부 프로세스 호출 (`child_process.exec("python -m khub.cli ...")`)
  - (b) QuickAdd 플러그인 — choice → script → CLI 호출
  - (c) Obsidian Shell Commands 플러그인 — 가장 직접적, but 외부 프로세스 권한
  - **결정 필요:** 어느 경로를 표준으로 할 것인가. 종일군의 "버튼 vs `/명령`" 선호도 확인 필요.
