## 7. 위상군 자가 반성

### Thin Router 체계의 효과
에이전트 8회 가동(탐색2 + 빌드리뷰4 + 기획1 + BUG-H 수정1)하면서 컨텍스트 상한에 도달하지 않음. Session 3 대비 개선.

### BUG-H 반복 수정의 교훈
PredictionBandSelector를 3번 재작성했다. 원인 체인:
1. `forwardToChart` isTrusted 문제 → pointer-events:none으로 전환
2. document-level 리스너에서 BAND_LIMITS.MIN_WIDTH_PCT undefined → NaN silent failure
3. pointer-events:auto로 재전환 → chart 네이티브 기능 전부 차단
4. pointer-events:none으로 복원 → 현재 상태

**근본 문제:** 빌드-리뷰 루프에서 기획군이 코드를 실제 실행 테스트하지 못함 (파일 리뷰만). BAND_LIMITS 속성 존재 여부 같은 런타임 문제를 기획 리뷰로 잡을 수 없음. 향후 빌드군에게 **vitest 단위 테스트 필수** 규칙을 강화해야 할 수 있음.

### 기획군 이벤트 설계 부재
TCL #25(인터랙션 분배 맵)가 이미 있었지만, 기획군이 ADR-004 기획서에서 구체적인 이벤트 매핑을 명시하지 않았음. §14를 하네스에 추가하여 보강.
