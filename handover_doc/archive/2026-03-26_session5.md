- 원인: 오버레이가 가격축 영역까지 덮어 좌클릭 가로채기
- 수정: `isInsideOverlay` → `isInsideSeriesPane` (`chart.timeScale().width()`로 가격축 제외)
- 파일: `PredictionBandSelector.tsx`

**Bug 3: 밴드 최소폭 UX (silent failure 제거)**
- 기획군 판단: 자동 스냅(center 기준 minWidth 확장) + 드래그 중 주황색 피드백
- 구현: `handleBandUp` 자동 스냅 + 3px 클릭 가드 + `isDragValid` IIFE + dragOverlay 색상 분기
- 파일: `PredictionBandSelector.tsx`, 기획서 `dnt_band_min_width_ux.md`

**Bug 4: 디버그 로그 제거**
- `useGameWebSocket.ts` line 65 `console.log` 삭제

**Bug 5: TimeHighlight 시간축 고정**
