### 4. 과녁 (PredictionTarget) 버그 수정
- **크기 배율 연동 제거:** `outerR = Math.abs(bottomY - topY) / 2` → `outerR = PREDICTION_TARGET.OUTER_RADIUS` (56px 고정)
- **절대 크기 확대:** 직경 112px (이모티콘 48px의 ~2.3배), 중심점 4px → 6px
- **X좌표 null 버그 수정:** `timeToCoordinate(미래시간)` → null 반환 문제 → `futureTimeToX()` (논리적 인덱스 기반) 사용
- **수정 파일:** PredictionTarget.tsx, prediction-target.ts(상수), GameScreen.tsx(klineCount prop 추가)

### 5. chartCoordinates 유틸리티 모듈 생성
- **파일:** `modules/chart/chartCoordinates.ts`
- **제공 함수:**

  - `futureTimeToX()` — 미래 시간 X좌표 (논리적 인덱스 기반)
  - `logicalIndexToX()` — 논리적 인덱스 → 픽셀
  - `priceToY()` — 가격 → Y좌표 (null 안전)

  - `isInViewportX()` / `isInViewportY()` — 뷰포트 체크
  - `dataPointToXY()` — 복합 좌표
