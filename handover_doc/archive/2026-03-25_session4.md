#### 조사 3: 독자 배포 방법
- **텔레그램 미니앱(TMA)이 게임체인저로 부상**
- PWA + TMA 듀얼 배포 권고

- 텔레그램 미니앱: 10억+ MAU, 크립토 네이티브, 심의 없음

#### 조사 4: 텔레그램 미니앱 기술/법률 심층
- 바탕화면 바로가기 가능 (addToHomeScreen API), 단 텔레그램 앱 경유
- 5분 대기(Notcoin)는 앱 문제이지 플랫폼 한계 아님 (권장 LCP 2.5초)
- DeviceStorage 5MB/유저, IndexedDB 수백MB 가능
- WebSocket, Canvas/WebGL 모두 지원 — 픽셀아트+차트에 충분
- TON 체인 필수 아님 (Stars만 쓰면 유저는 크립토 불필요, 개발자 인출만 TON)
- 앱스토어 규제(Simulated Gambling, 연령등급, 크립토정책) → TMA에서 모두 사라짐
- **한국법(사행성, 유사투자자문업, 개인정보보호법)은 플랫폼 무관 적용**

#### 조사 5: 한국 크립토 vs 주식 참여자 상세
- 크립토 거래소 1,629만 > 주식 1,423만 (2025.02 역전)

- 20대 남성 크립토 투자자만 216만명
- 67%가 50만원 미만 소액 투자자
- 한국 텔레그램 DAU 116~152만, 크립토 커뮤니티에서 필수 플랫폼

#### 조사 6: Notcoin/Hamster BM + 한국 TMA/PWA 현황
- **Notcoin/Hamster 모두 2024 피크 후 급격 쇠퇴** (Hamster -96%)
- 근본 원인: tap-to-earn 게임성 한계(60%) + 토큰 에어드롭 구조(30%)
- 한국 텔레그램 점유율 ~1.1% (카카오톡 94.4%) → **한국-Only TMA는 시장 극히 좁음**
- 한국 타겟 TMA 성공 사례 없음, 한국 PWA 성공 사례 없음

#### 조사 7: 6대 핵심 질문 통합 리서치 (최종 교정)
- **ARPU $271.50 과대 확인** — RPG 편중, DnT 현실은 $20~50/년

### 3.2 DnT v3 최종 포지셔닝 (종일군 인사이트)
**"크립토 개미의 집단 센티먼트 도구 + 중독성 분봉 예측 게임"**

네트워크 효과 구조:
```
[솔로 가치] ← 3~5분 분봉 예측 중독성 (1명이라도 재밌음)
     ↓
[집단 가치] ← 히트맵이 의미 있는 센티먼트가 됨
     ↓
[네트워크 효과] ← "참여자↑ = 내 정보 가치↑" → 자발적 초대
     ↓
[미디어 가치] ← "10만 개미의 BTC 예측" = 뉴스/SNS 인용 가능
```

### 3.3 확정된 사항 종합

| 항목 | 확정 내용 |
|------|----------|
| 배포 | TMA(원본, 최대 자극) + PWA(보조) → 앱스토어(순화 이식) |
| 시장 | 글로벌 우선 (영어), 한국어는 PMF 후 추가 |
| 자산 | 크립토 5종(BTC/ETH/XRP/SOL/DOGE) + 미국주식 5종(TSLA/NVDA/AAPL/PLTR/GOOGL) |
| 한국주식 | 한국어 버전 추가 시 포함 (삼성전자/SK하이닉스/LG에너지솔루션/하이브/삼성바이오) |
| BM | 광고(60~70%) + Stars IAP(20~30%) + PWA 직접결제(보조). 토큰 없음 |
| 포지셔닝 | 크립토 개미 집단 센티먼트 도구 + 분봉 예측 중독 게임 |
| 예측 주기 | 3~5분 분봉 (솔로 중독) + 일봉 (미국주식 시차 활용) |
| 리텐션 | 분봉 중독(즉각 보상) + 시차 3포인트(아침/오후/밤) + 히트맵(네트워크) |

| 코스메틱 BM | 한국 유저 약점이나 "지위(status)" 결합 시 기회 (트렌드 이동 중) |

## 7. 종일군 인사이트 메모

- **"TMA에 가장 자극적인 형태로 글로벌 런칭 → 앱스토어 순화 이식"** — 듀얼 버전 전략의 원안. 원본이 자극적이고 이식본이 순화라는 발상의 전환
- **"전세계 크립토 개미참여자들이 대규모 크립토 작전을 위한 툴"** — DnT의 핵심 네트워크 효과 메커니즘. "참여자↑ = 내 정보 가치↑" → 자발적 바이럴
- **"3~5분마다 한번씩 가격대를 예측하는 자극, 중독 솔로플레이 재미"** — 콜드스타트 해결의 열쇠. 히트맵 없이도 솔로 가치 제공
- **"시차 발생으로 하루에 두 번은 확인해볼 수 있으니까"** — 조사 결과 3회까지 확장 가능. 시장 이벤트 자체가 접속 동기
- **"현재 시장조사 에이전트는 전문가인가?"** — AI 리서치의 한계를 정확히 인식. 1차 스크리닝으로 활용하고 최종 판단은 인간이 하는 원칙 재확인
