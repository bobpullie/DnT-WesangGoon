## 완료 사항

### 1. ADR-012 설계 + 3 Phase 빌드

**ADR-012:** `docs/decisions/ADR-012_m2_phase0_1280x720_layout.md`
**기획서:** `docs/planning/DnT_M2_Phase0_Design_1280x720.md` (기획군 작성, 1000줄)

#### Phase 1: 레이아웃 골격
동적 레이아웃 → 고정 레이아웃 전면 전환:
- `GameScreen.tsx` 전면 재구성: NavRail | TopHUD + chartArea + BottomBar | SidePanel
- `NavRail.tsx` 수정: 40px 유지, 상시 표시, `disabled` prop (LIVE/LOCKED 비활성 + 토스트)
- `TopHUD.tsx` 신규: BTC 가격 + 상태 뱃지(색상별) + 타이머 + 연결 점
- `SidePanel.tsx` 신규: 상시 고정 셸 (상단 가변 + 하단 UserCard 고정)
- `UserCard.tsx` 신규: 닉네임/레벨/스코어/적중률 상시 표시
- `BottomBar.tsx` 신규: 기존 bottomBar 로직 이전
- `App.tsx` 수정: GameScreen이 자체 레이아웃이므로 game 페이지에서 App-level NavRail 제거
- `LAYOUT` 상수 추가 (shared/constants/game.ts)

**기획군 검토:** CONDITIONAL PASS → LAYOUT 수치 B옵션으로 확정 (종일군 승인)

#### Phase 2: 패널 콘텐츠
5개 상태별 사이드패널 콘텐츠 + P&L 이중 표시:
- `SideBrowsing.tsx` — 이전결과 + 세션통계 + PREDICT NOW CTA (BottomBar에서 이전)
- `SidePredicting.tsx` — 카운트다운 + 과녁 상태 + 밴드 상한/하한 + SUBMIT (골드)
- `SideLocked.tsx` — LOCKED IN(시안) + P&L 방향 + Entry + 카운트다운
- `SideLive.tsx` — P&L 대형 게이지(20px) + 데미지/힐 카운터 + 밴드 점수 + CC + 시간 진행
- `SideResult.tsx` — ScoreAnimation + CC 분해 + nearMiss + P&L 최종 + 액션 버튼
- `PnLCompact.tsx` — 차트 좌상단 P&L 축약 오버레이 (AutoStakeGauge 대체, left:16 top:16)
- BottomBar에서 PREDICT NOW 제거 (사이드패널로 이전)

**기획군 검토:** CONDITIONAL PASS — MINOR 8건 (최근 3턴, 공유 버튼 등 후속 이터레이션)

#### Phase 3: 이펙트 & 폴리시
- `DamagePopup.tsx` — CSS keyframe 팝업 (floatUp + shake, 임계값/빈도 제한, 동시 3개)
- `SideLive.tsx` — 데미지/힐 누적 실제 카운터 (P&L 변동 추적)
- `SidePanel.tsx` — crossfade 트랜지션 (roundState 전환 시 opacity 토글)
- `PnLCompact.tsx` — scale(0→1) 진입 애니메이션 (0.3s ease-out)
- `GameScreen.tsx` — resultDimOverlay 강화 (0.6 + transition)
