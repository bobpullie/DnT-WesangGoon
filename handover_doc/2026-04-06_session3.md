## 완료 사항

### 1. 블로커 1: Gemini API 429 — 해결
- **근본 원인 1:** Windows 시스템 환경변수에 이전(소진된) API 키 `AIzaSyAoqFkA...`가 남아있어 `.env` 파일의 새 키를 덮어씀
  - Docker Compose는 시스템 env > .env 파일 순서로 우선
  - `docker compose restart`는 환경변수 갱신 안 함 → `down` + `up` 필요
- **근본 원인 2:** Gemini 2.5 Flash 무료등급은 **RPD 20회** (하루 20회 요청)
  - OpenClaw 재시도(4회/실패) + 디버깅 curl로 즉시 소진
  - **결제 활성화 무료체험** 프로젝트는 RPD 500 / RPM 1,000
- **해결:** 종일군이 결제 활성화된 Google Cloud 프로젝트의 API 키 `AIzaSyBN...` 제공
- **조치:** `.env` + Windows 사용자 환경변수 모두 업데이트, `docker compose down && up`

### 2. 블로커 2: Ollama delivery-mirror 타임아웃 — 해결
- **근본 원인 1:** `gateway.mode` 미설정 → delivery-mirror 로컬 라우팅 실패
- **근본 원인 2:** `api: "openai-completions"` 사용 → OpenClaw에 네이티브 `api: "ollama"` 지원됨
  - baseUrl도 `/v1` 제거 필요 (`http://host.docker.internal:11434`)
- **근본 원인 3:** 타임아웃이 `rate_limit`으로 오분류 → 세션에 쿨다운 캐시
  - `models.json`, `sessions/*.jsonl`, `sessions.json` 삭제로 초기화 필요
- **OpenClaw 알려진 버그:** #61487 (하드코딩 60초 타임아웃), #5240/#13336 (rate_limit 오분류), #7725 (컨텍스트 주입 블로트)
