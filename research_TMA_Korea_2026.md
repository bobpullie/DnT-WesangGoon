# 텔레그램 미니앱(TMA) 생태계 & 한국 시장 리서치 리포트

> 조사일: 2026-03-25 | 조사 에이전트: 위상군
> **주의:** PWA(Progressive Web App)와 TMA(Telegram Mini App)는 완전히 다른 개념이다. 본 리포트에서는 명확히 구분하여 기술한다.

## 1. Notcoin 상세 분석

### 1.1 유저 규모

| 지표 | 수치 | 시점 |
|------|------|------|
| 누적 플레이어 | 3,500만+ | 2024 피크 |
| 현재 상태 | 활성 유지 (축소) | 2026.03 |
| NOT 시가총액 | ~$39M~$54M | 2026.03 |
| NOT 토큰 가격 | $0.00039~$0.00041 | 2026.03 |

- Notcoin + Hamster Kombat 합산 피크: **3억 명 이상**
- MAU/DAU 최신 데이터는 비공개이나, TMA 생태계 전체가 2024.09 피크(14.4억 MAU) 이후 **150~190M MAU로 안정화** (2025 중반 기준)

### 1.2 비즈니스 모델 구조

```
┌─────────────────────────────────────────────────┐
│              Notcoin BM 구조                      │
├─────────────────────────────────────────────────┤
│                                                   │

│                                                   │
│  ┌──────────────┐    ┌──────────────────┐         │
│  │ Tap-to-Earn  │───>│ NOT 토큰 획득    │         │
```

**핵심 수익 구조:**
- **토큰 자체가 비즈니스 모델**: 전체 공급량의 **78%를 커뮤니티에 분배** (공정 출시 모델)
- **에어드롭 허브**: 신규 TON 프로젝트들이 NOT 홀더에게 에어드롭 → Notcoin이 유통 플랫폼 역할
- **NOT Games 플랫폼**: 서드파티 게임 퍼블리싱 → "소셜 바이럴 게임의 넷플릭스" 지향
- **직접적 광고/인앱결제 수익 모델은 확인 불가** — 전통적 BM이 아닌 **토큰 경제 기반**

**NOT 토큰 경제:**
| 할당 | 비율 |
|------|------|
| 초기 마이너 & 바우처 홀더 | 78% |
| 신규 유저/트레이더/개발 | 22% |
| 총 공급량 | 1,027억 NOT |
| 특이점 | Day 1에 100% 언락 (선형 베스팅 없음) |

### 1.3 게임 구조

| 게임/모드 | 설명 |
|-----------|------|
| **Core Tap-to-Earn** | 화면의 코인을 탭 → 1탭 = 1 Notcoin, 에너지 시스템 |
| **Explore** | dApp/프로젝트 참여로 보상 획득 (2024.07~) |
| **VOID** | 실시간 전략 게임 (스킬 기반, 2025.03~) |
| **Lost Dogs** | 스토리 기반 퀘스트 게임 ($NOT + $WOOF 듀얼 토큰) |
| **리더보드** | 일간/전체 이중 리더보드 시스템 |
| **부스터/터보 모드** | 수익 가속 아이템 |

### 1.4 한국 유저

**팩트:** 한국 유저 비율/규모에 대한 공개 데이터 없음.
**추정:** 한국 텔레그램 메신저 점유율 ~1.1% (카카오톡 94.4%) → Notcoin 한국 유저는 극소수로 추정.

### 1.5 현재 상태 (2026.03)

- **쇠퇴 단계이나 완전 소멸은 아님**
- NOT 토큰: ATH 대비 대폭 하락, 시가총액 $39~54M
- NOT Games 플랫폼으로의 전환 시도 중 (단순 탭 게임 → 게임 퍼블리싱 플랫폼)
- "소셜 바이럴 게임의 넷플릭스"라는 비전은 아직 검증 안 됨

## 4. 한국에서 텔레그램 미니앱(TMA) 현황

### 4.1 한국 텔레그램 기본 현황

| 지표 | 수치 | 비고 |
|------|------|------|
| 한국 메신저 점유율 | **~1.1%** | 카카오톡 94.4%가 압도적 1위 |
| 한국 텔레그램 사용 목적 | 크립토 커뮤니티, 해외 연락, 프라이버시 | 일반 메신저로는 극소수 사용 |
| 한국 대상 텔레그램 코인 채널 | **500개+** | 크립토 정보 교류가 핵심 용도 |
| 한국 소셜미디어 조사 포함 여부 | **포함되지 않음** | 2024 KPF 조사에서 텔레그램 미언급 |

### 4.2 한국인이 많이 사용하는 TMA

**팩트:** 한국인이 많이 사용하는 TMA에 대한 공개 데이터 없음.
**추정:**
- 한국 텔레그램 사용자의 대다수가 **크립토 커뮤니티 채널** 사용자
- Notcoin/Hamster Kombat 등에 참여한 한국인이 있으나 규모 미공개
- 한국 시장을 직접 타겟한 TMA **성공 사례는 확인 불가**

### 4.3 한국 크립토 유저 — TMA의 잠재 시장

| 지표 | 수치 | 시점 |
|------|------|------|
| 암호화폐 거래소 계정 수 (중복 포함) | **2,444만 개** | 2025 |
| 실질 거래 가능 이용자 | **1,077만 명** | 2025.06 |
| 코인판 MAU | **530만 명** | 2023 말 |
| 한국 글로벌 현물거래 비중 | **16%** | 2025 |

**핵심 인사이트:**
- 한국 크립토 유저는 **1,000만+ 명**으로 세계 최대급
- 그러나 이들의 커뮤니케이션은 **카카오톡 오픈채팅, 디시인사이드, 코인판**이 중심
- 텔레그램은 "정보 수신용" (채널 구독)이지, "앱 실행 플랫폼"으로는 인식되지 않음
- **한국에서 TMA는 사실상 미개척 시장**

### 4.4 한국 TMA 인지도/사용률 평가

| 항목 | 평가 |
|------|------|
| TMA 개념 인지도 | **극히 낮음** — 크립토 얼리어답터 외 인지 사실상 제로 |
| TMA 사용 경험 | **극소수** — 에어드롭 목적의 일회성 참여 추정 |
| TMA 반복 사용 | **사실상 없음** — 한국용 킬러 TMA 부재 |
| 성장 잠재력 | **중-하** — 카카오톡 미니앱 생태계가 이미 존재 |

## 5. 한국에서 PWA 현황 (TMA와 별도)

### 5.1 한국 PWA 성공 사례

**팩트:** 한국 시장에서 PWA로 성공한 뚜렷한 국내 서비스 사례는 확인 불가.

**글로벌 PWA 성공 사례 (참고):**
| 서비스 | 성과 |
|--------|------|
| Starbucks | DAU 2배 증가, 앱 크기 148MB → 233KB (99.84% 감소) |
| Twitter Lite | 트윗 작성 80% 증가, 이탈률 20% 감소 |
| Flipkart Lite | 전환율 70% 상승 |
| Alibaba | 전환율 76% 증가 |
| Pinterest | 광고 수익 44% 증가 |

### 5.2 한국 유저의 PWA 인지도

| 항목 | 상태 |
|------|------|
| PWA 개념 인지도 | **매우 낮음** — 개발자 외 일반인은 모름 |
| "홈 화면에 추가" 경험 | **극소수** |
| 앱 설치 = 스토어 인식 | **강함** — "앱 = Play Store/App Store에서 설치"가 한국의 기본 인식 |
| iOS Safari PWA 지원 | 2023.03~ (iOS 16.4) Web Push 지원으로 개선 |

### 5.3 한국 주요 서비스의 PWA 현황

- **배달의민족, 쿠팡, 네이버, 카카오:** 모두 **네이티브 앱 중심** 전략. PWA 별도 제공 미확인
- 한국 대기업 서비스들은 **앱스토어 랭킹, 푸시 알림, 결제 연동** 등의 이유로 네이티브 앱을 유지
- 카카오엔터테인먼트 기술 블로그에서 A2HS(Add to Home Screen) 기술 문서를 발행한 바 있어, 일부 기술적 관심은 존재

### 5.4 PWA vs 네이티브 앱 — 한국 시장 특성

| 요인 | 한국 시장 특성 | PWA에 미치는 영향 |
|------|--------------|-----------------|
| 스마트폰 보급률 | 95%+ (세계 최고 수준) | 네이티브 앱 접근성 이미 높아 PWA 필요성 낮음 |
| 통신 인프라 | 5G 보급률 세계 1위급 | 오프라인 캐싱 필요성 낮음 |
| 앱 스토어 문화 | 매우 강함 (Play Store 중심) | "스토어 밖 앱"에 대한 거부감 |
| 결제 생태계 | 카카오페이, 네이버페이, 삼성페이 | 네이티브 앱과 깊게 통합, PWA에서 제한적 |
| 사용자 행동 | 슈퍼앱(카카오, 네이버) 안에서 대부분 해결 | PWA보다 슈퍼앱 미니앱이 자연스러운 경로 |

**PWA 글로벌 시장 규모:** 2025년 $52.3억 초과 예상 (Straits Research)
**한국에서의 위치:** 글로벌 트렌드와 괴리 — 네이티브 앱 선호도가 극히 강한 시장

## 6. DnT에 대한 시사점

### 6.1 DnT가 TMA로 한국 시장을 타겟할 때의 현실적 기대치

**냉정한 평가:**

| 요소 | 현실 |
|------|------|
| 한국 텔레그램 MAU | 추정 200~500만 (메신저 점유율 1.1% 기반) |
| 그 중 TMA 인지/사용 | 극소수 (크립토 유저 일부) |
| DnT의 TMA 한국 타겟 TAM | **수만~수십만 명 수준** (매우 낮음) |
| 바이럴 성장 가능성 | **낮음** — 한국 바이럴은 카카오톡/인스타 기반 |

**그러나 고려할 점:**
1. **한국 크립토 유저 1,000만+** 중 텔레그램 채널 구독자가 상당수 존재
2. 이들은 이미 텔레그램 앱을 설치하고 있어, TMA 접근 장벽이 없음
3. "주식 가격 예측 게임"이라는 DnT 컨셉은 **한국 크립토 유저의 핵심 관심사**(가격 예측)와 직결
4. 토큰 없이도 Stars 기반으로 운영 가능하므로 **규제 리스크 회피** 가능

### 6.2 토큰 없는 TMA의 가능성 (Stars + 인앱결제만)

**장점:**
| 항목 | 설명 |
|------|------|
| 규제 리스크 | **제로** — 토큰 발행 없으므로 한국 가상자산법 적용 안 됨 |
| 개발 복잡도 | **낮음** — 스마트 컨트랙트/토큰노믹스 불필요 |
| 수익 실현 | **즉시** — Stars 결제는 실시간 정산 가능 |
| 사용자 경험 | **매끄러움** — Apple Pay/Google Pay 원탭 결제 |
| 텔레그램 수수료 | **거의 없음** — 제3자 결제 수수료만 |
| iOS/Android 호환 | **완전 준수** — 앱스토어 정책 위반 없음 |

**단점:**
| 항목 | 설명 |
|------|------|
| 바이럴 동력 | **약함** — 토큰 에어드롭 없으면 "무료로 돈 벌기" 유인 없음 |
| 초기 유저 확보 | **자체 마케팅 필요** — 에어드롭 바이럴 불가 |
| CAC | TMA 자체 CAC는 낮으나, 한국 타겟 유저 도달 비용은 별도 |

**현실적 수익 시뮬레이션:**
```
DAU 5,000명 기준:
- 광고 수익: 5,000 DAU × 2회 광고 × $2 CPM ÷ 1,000 = $20/일 = $600/월
- Stars IAP: 5,000 DAU × 5% 결제율 × $1 평균 결제 = $250/일 = $7,500/월
- 합산: ~$8,100/월

DAU 50,000명 기준:
- 광고: ~$6,000/월
- Stars IAP: ~$75,000/월
- 합산: ~$81,000/월
```

### 6.4 최종 권고

**DnT TMA 한국 시장 전략 요약:**

1. **한국만 타겟하면 시장이 너무 작다.** 한국 텔레그램 TMA 사용자는 수만 명 수준. 글로벌(특히 CIS/동남아/중동) + 한국을 동시 타겟해야 함.

2. **토큰 없는 Stars+광고 하이브리드가 현실적이다.**
   - 한국 가상자산 규제(VASP 등록, 트래블 룰)를 피할 수 있음
   - 개발 리소스를 게임 자체에 집중 가능
   - 수익 실현이 즉각적

3. **"주식/코인 가격 예측"은 TMA에서 희소한 장르다.**
   - 현재 TMA는 탭-투-언/캐주얼 게임이 90%+
   - 가격 예측 게임은 크립토 유저의 핵심 관심사와 정확히 매칭
   - **기대심리 히트맵**은 차별화 킬러 피처가 될 수 있음

4. **한국 크립토 유저 유입 경로는 "텔레그램 코인 채널 바이럴"이 가장 현실적이다.**
   - 500개+ 한국 코인 채널에 DnT 봇/미니앱 링크를 유통
   - "이번 주 BTC 가격 맞추기 챌린지" 같은 무료 참여 이벤트로 초기 유저 확보
   - 리더보드 경쟁심리가 바이럴 트리거

5. **기대 규모: 현실적으로 한국 DAU 1,000~5,000명이 초기 목표.**
   - 글로벌 합산 DAU 10,000~50,000 달성 시 월 $8K~$80K 수익 가능
   - 한국은 전체의 10~20% 비중으로 시작
