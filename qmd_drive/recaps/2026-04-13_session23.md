# Session 23 — Atlas src-layout Migration & Skill Packaging
> 2026-04-13 | 위상군 | E:\DnT\DnT_WesangGoon

## 핵심 교훈
- src-layout 마이그레이션 시 `pythonpath` 변경만으로는 부족 — 하드코딩된 `Path(__file__).parent.parent`, hook template 내 import 경로, 테스트 내 상대경로도 함께 갱신 필요
- `importlib.resources.as_file()` context manager는 Windows에서 필수 — 직접 Path로 접근하면 zip 패키지에서 실패
- Code review에서 argparse default=None vs default="." 차이가 Critical 버그로 식별됨 — 얇은 CLI 래퍼도 리뷰 가치 있음
- Subagent-driven development: 구조 마이그레이션(tightly coupled tasks)은 하나의 서브에이전트로 배치, TDD 신규 기능은 별도 서브에이전트가 효율적
