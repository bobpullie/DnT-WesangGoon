## 1. 3자산 클래스 거래 시간 매핑 (KST 24시간 타임라인)

### 1.1 크립토 (24시간)

크립토는 24/7 거래되지만, 시간대별 변동성 특성이 뚜렷하다.

| KST 시간대 | 글로벌 세션 | 변동성 특성 |
|:----------:|:----------:|:-----------|
| 00:00~07:00 | 아시아 심야~유럽 오후 | 변동성 최저 구간 (UTC 03:00~04:00 부근이 하루 중 가장 낮음) |
| 07:00~15:00 | 아시아 주간 | 아시아 시장 개장으로 거래량 점진 상승, 변동성 소폭 증가 |
| 15:00~19:00 | 유럽 개장~오전 | 유럽 시장 참여로 거래량 급증, 변동성 상승 |
| 19:00~01:00 | 유럽+미국 겹침 | **변동성 최고 구간** — 양 시장 겹침으로 거래량 정점 |
| 22:30~05:00 | 미국 정규장 | 미국 매크로 뉴스 발표(고용, CPI 등)에 크립토도 급반응 |

**핵심 인사이트:** 비트코인의 시간별 거래량은 reverse V-shape 패턴을 보이며, **유럽+미국 세션 겹침 시간(KST 19:00~01:00)**에 최고치를 기록한다. 아시아 시장 개장은 변동성에 미미한 영향을 준다 (ScienceDirect 연구).

### 2.2 핵심 접속 포인트 (하루 최소 2~3회)

종일군의 인사이트 "시차 발생으로 하루에 두 번은 확인해볼 수 있으니까"를 구체화하면:

1. **Morning Check (07:00~09:00):** 미장 결과 확인 + 한국장 개장 예측 제출
2. **Afternoon Check (15:00~16:00):** 한국장 결과 확인 + 보상 수령
3. **Night Check (22:00~23:00):** 미장 개장 예측 제출 + 크립토 변동성 관전

이 3포인트 구조는 **아침-오후-밤** 리듬으로, 유저의 자연스러운 생활 패턴과 일치한다.

### 2.3 리텐션 효과 분석

**연구 근거:**
- 일일 챌린지를 제공하는 게임은 참여 유저의 **65%가 매일 접속**한다 (Pushwoosh)
- 판타지 스포츠 앱에서 실시간 업데이트가 **유저 참여의 #1 이유** (78%, Fantasy Sports & Gaming Association)
- 소셜 기능을 결합한 게임은 DAU **25% 증가** (Mistplay)
- 복수 터치포인트를 제공하는 앱은 단일 터치포인트 대비 리텐션 유의미하게 상승

**DnT 시차 활용의 차별점:**
- 기존 가격 예측 앱은 단일 자산 클래스 → 하루 1회 접속
- DnT는 3자산 클래스의 시차를 활용 → **하루 2~3회 자연스러운 접속 동기**
- 이는 인위적인 "일일 보상" 메커니즘이 아니라, **시장 이벤트 자체가 접속 동기**를 만드는 구조

### 4.1 선정 기준

| 기준 | 가중치 | 근거 |

| MZ세대 관심도 | 높음 | 핵심 타겟 유저 |
| 일봉 변동성 | 중간~높음 | 대형주 기준 일 1~5% 변동 |

- **공공데이터포털** (data.go.kr): 금융위원회 "주식시세정보" API — 일봉 OHLC 제공, 무료, 일 1회 갱신 (영업일 D+1 오후 1시)
- **한국투자증권 KIS Open API**: REST 방식, 일봉/주봉/월봉 OHLC 실시간 조회 가능, 무료 계정 지원
- **KRX 정보데이터시스템** (data.krx.co.kr): 한국거래소 공식 데이터

## 6. 시차 활용 리텐션 사례

### 6.1 판타지 스포츠 리그의 시차 활용

**DraftKings / FanDuel:**
- NFL, NBA, MLB, 축구 등 복수 스포츠 리그의 시간대 차이를 활용
- NFL(일요일 오후) → NBA(매일 저녁) → MLB(매일 오후~저녁) → 유럽축구(한국 시간 밤)
- 유저는 하루 중 여러 스포츠 이벤트에 참여하며 복수 접속
- **결과:** 일일 무료 풀(Daily Free Pool)이 캐주얼 유저와 하드코어 팬 모두의 정기적 참여를 유도

### 6.2 크립토 + 주식 혼합 앱

- 2025~2026년 크립토-주식 겸용 브로커리지(Robinhood, eToro 등)가 "24시간 거래" 를 마케팅 포인트로 활용
- 주식시장 폐장 시 크립토로 자연스럽게 관심 전환 → **세션 간 전환(Session Bridging)** 효과

### 6.3 "하루 여러 번 접속" 설계의 리텐션 효과 연구

| 전략 | 효과 | 출처 |
|:----:|:----:|:----:|
| 일일 챌린지 (Daily Reset) | 참여 유저 65% 매일 접속 | Pushwoosh |
| 실시간 업데이트 제공 | 78% 유저가 #1 참여 이유로 꼽음 | FSGA |
| 소셜 기능 결합 | DAU 25% 증가 | Mistplay |
| 개인화된 경험 | 리텐션 최대 20% 향상 | ContextSDK |
| 다중 터치포인트 푸시알림 | D1 리텐션 29.46% → D7 8.7% (업계 평균 대비 상회) | Segwise |

### 6.4 DnT의 차별적 리텐션 구조

기존 예측 게임/앱과 DnT의 핵심 차이:

```
기존 앱:  [접속] ──── 24시간 대기 ──── [접속]       → D7 리텐션 ~8%
          단일 자산, 하루 1회 이벤트

DnT v3:   [아침] ─── 6h ─── [오후] ─── 7h ─── [밤]  → D7 리텐션 목표 15%+
           한국장       한국장 마감      미국장
           + 크립토      + 크립토        + 크립토
```

**핵심 메커니즘:** "기다림"이 아니라 "다음 시장 이벤트까지의 카운트다운"이 접속 동기가 된다.
